== Redmine/Chiliproject Companies Plugin

This plugin allows the redmine/chiliproject administrator to create companies and link them to users and projects.

== Installation

1. Copy the plugin directory into the vendor/plugins directory

2. Migrate plugin:
   rake db:migrate_plugins

3. Start Redmine

Installed plugins are listed and can be configured from 'Admin -> Plugins' screen.
